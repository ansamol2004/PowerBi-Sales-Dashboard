# 🛒 Supermarket Sales Dashboard (Power BI)

## 📌 Project Overview
This Power BI dashboard analyzes supermarket sales across cities, products, and managers. It helps track performance, identify trends, and support data-driven decisions.

## 📊 Key Features
- Sales analysis by city and product
- Quantity analysis across locations
- Sales trend over time
- Manager performance evaluation
- Payment method insights
- Interactive filters and navigation

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
